
1. Download Python at https://www.python.org
2. Install
	a. If prompted, check 'ADD TO PATH'
3. Open CMD at the script's directory
4. Run this command on CMD (don't close CMD): pip install -r requirements.txt
5. Open config.json and enter your info. DO NOT MODIFY THE FORMAT! Save the file and close.
6. Run this command on CMD: python3 ./vaccine_passport.py
